Per compilar i executar el programa, fer "make" en la carpeat "Code", i executar l'exactuable que es generarà amb el nom:
SuperSopa.exe

Quan s'executi el main s'obrirà un menu amb 3 opcions principals:
1) Generate words file   // Es demanen paràmetres com el nombre de paraules i la seva longitud màxima i mínima              
2) Generate board file   // Es demanen paràmetres com les dimensions del tauler
                                                        
3) Call solver  // Es demanen els noms dels arxius de paraules i de tauler generats anteriorment, i s'obre el següent menu:

What algorithm do you want to use?
1) Patron Solver // Abre el menu de selección del diccionario para usar el patron solver
2) Word Solver  // Pide archivo de salida y soluciona el problema con el word solver

What type of dictionary do you want? // Solo para el Patron Solver
1) Tree dictionary
2) Hash dictionary
3) Vector dictionary

Una vez seleccionada una opción, se pide el archivo de salida y se soluciona el problema.
